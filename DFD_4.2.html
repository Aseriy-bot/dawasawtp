Help & Support

Learn how to use the DAWASA Flowmeter Dashboard effectively

Getting Started

Quick introduction to the dashboard

Welcome to the DAWASA Flowmeter & Pressure Monitoring Dashboard. This system provides real-time monitoring of water flow and pressure data across multiple sites and pipe sizes.

  • Dashboard View: See all 5 flowmeters and pressure transmitters at a glance
  • Individual Pages: Click on any meter to view detailed metrics, charts, and historical data
  • Real-time Updates: Data refreshes automatically at your configured interval
  • Theme Selection: Choose between light and dark modes in Settings

Navigation Guide

Understanding menu items and structure

Main Menu Items:

  • Dashboard: Overview of all flowmeters and pressure transmitters
  • Flowmeter Pages: Individual 36", 48", 24", 30", and 16" pipe flow monitoring
  • Pressure Pages: Individual pressure readings for each pipe section
  • Alerts: Active and historical alert notifications
  • Historical Data: View and export data with filters
  • Settings: Configure system preferences and units

Metrics Explained

Understanding the data you see

Metric Description
Latest Flow/Pressure Current real-time reading from the sensor
Peak Flow/Pressure Highest value recorded during the monitoring period
Average Flow Rate Mean flow rate calculated from all readings
Total Flow Today Cumulative flow volume measured in current day

Key Features

Powerful tools at your fingertips

  • Real-time Charts: Interactive line charts showing flow/pressure trends
  • Status Indicators: Color-coded dots show system status (Green=Good, Yellow=Warning, Red=Critical)
  • Data Export: Export historical data as CSV or PDF for analysis
  • Search Functionality: Quickly find sites, pipes, and data values
  • Alert Management: View, filter, and clear system alerts
  • Multi-site Monitoring: Monitor RPC, Kibaha, and Mlandizi sites simultaneously

Troubleshooting

Solutions to common issues

  • No Data Showing? Check your internet connection and ensure MQTT broker is connected
  • Alerts Not Updating? Verify alert notification settings are enabled in Settings
  • Export Not Working? Ensure at least one data point exists and filters are not too restrictive
  • Theme Not Changing? Clear browser cache and refresh the page after changing theme
  • Charts Loading Slowly? Reduce data retention period in Settings or clear cache
